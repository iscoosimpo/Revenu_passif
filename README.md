# Site de vente - English Mastery Book

Landing page premium pour vendre votre livre depuis votre audience TikTok.

## Fichiers

- `index.html` : structure complète de la page
- `styles.css` : design responsive (mobile + desktop)
- `script.js` : interactions simples (annee auto, bouton d'achat)

## Lancer le site

Ouvrir simplement `index.html` dans un navigateur.
